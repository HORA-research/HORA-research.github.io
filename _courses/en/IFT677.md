## Course Overview

This course provides a comprehensive introduction on technological choices that govern the implementation of database management systems.
Students will:

1. implement an artifact management system (sources, versions, configurations, documentation, etc.) for artifacts produced during the development and evolution of software systems that include databases;
2. implement a continuous integration mechanism for software systems that include databases;
3. develop a machine-to-machine interface that incorporates an access policy compliant with cybersecurity policies;
4. optimize the representation of logical and physical database models;
5. optimize query execution based on execution plans and usage context;
6. implement a system for both batch and continuous data loading;
7. implement a distributed database management system.
