## Course Overview

This course provides a comprehensive introduction to database principles and practices. 
Students will:

1. use relational theory to describe a data structuring problem;
2. model data using an entity-association (EA) diagram, a UML class diagram, and the relational model;
1. transform an EA model and a UML class diagram into a relational model;
2. normalize a database to FNBC and 5FN;
3. define, using SQL, the domains, types, relationships, constraints procedures, and automations required to define a medium-sized database;
1. query and update a relational database using SQL;
2. apply best practices for defining and managing keys;
3. integrate all the concepts presented to model a data management problem and implement this model using SQL;
1. identify the basic components of a relational database management system.
