## Course Overview

This course provides a comprehensive introduction to database principles and practices. 
Students will:

1. model, design, and justify the choices underlying the implementation of an operational (transactional) database in accordance with best practices;
2. model, design, and account for the choices underlying the implementation of an analytical database in accordance with best practices;
3. apply the main temporal models to concrete problems;
4. apply the main historicization models to concrete problems;
5. apply the main graph models to concrete problems.
